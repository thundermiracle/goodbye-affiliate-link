# Goodbye Affiliate Link

ページ中のアフィリエイトリンクを元々のURLに戻すChrome拡張機能です。

## 機能

- ページ中のアフィリエイトリンクを検出します
- 検出したリンクを元々のURLに戻します

## 開発

- TypeScriptとReactを使用しています
- [WXT](https://wxt.dev)フレームワークを使用して拡張機能を構築しています

```shell
pnpm install
pnpm run dev
```
